## Registering a new project

1. Run `.claude/new-project-start.md` fully
2. Add a row to this table
3. Add a row to the prefix registry in `leanspirited-standards/standards/backlog-format.md`
4. Commit both files

## Prefix conflicts

Check this table before assigning a new prefix.
Prefixes are permanent — once assigned to a project, never reassign.
